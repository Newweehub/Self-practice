1.
1.1
- ใช้ .filter() เพื่อหาคำที่มีคำว่า "mango" อยู่ข้างใน ด้วย includes()
- ผลลัพธ์: ["mango", "mangosteen", "mangoes"]
1.2
- ใช้ .map() เพื่อเปลี่ยนทุกคำใน words ให้เป็นตัวพิมพ์เล็ก ด้วย .toLowerCase()
- ผลลัพธ์: ["mango", "apple", "mangosteen", "orange", "mangoes"]
1.3
- ใช้ .map() แปลงทุกคำให้เป็นตัวพิมพ์ใหญ่ก่อน
- แล้วใช้ .filter() เลือกคำที่มีความยาวน้อยกว่า 7 ตัวอักษร
- ผลลัพธ์: ["MANGO", "APPLE", "ORANGE"]

2.
2.1
- ใช้ .sort() เรียงลำดับตัวอักษร A -> Z
- ใช้ .reverse() กลับลำดับเป็น Z -> A
- ผลลัพธ์: [ 'orange', 'mangosteen', 'mangoes', 'mango', 'Apple' ]
2.2
- ใช้ .sort() เรียงลำดับ object students ตามชื่อ (name) แบบตัวอักษร
- ผลลัพธ์: students เรียงตามชื่อจาก A -> Z

3.
3.1
- ใช้ .find() หาคำแรกที่มี "apple" อยู่ข้างใน (ไม่สนพิมพ์เล็กพิมพ์ใหญ่)
- ผลลัพธ์: "Apple"
3.2
- ใช้ .find() หานักเรียนที่ id เท่ากับ 2
- ผลลัพธ์: { id: 2, name: 'Suda Deejai' }

4.
4.1
- ใช้ .reduce() บวกจำนวนสินค้า (quantity) ของทุกชิ้นรวมกัน
- ผลลัพธ์: 15
4.2
- ใช้ .reduce() เอาราคารวมทั้งหมด / จำนวนสินค้าใน array (.length) -> หาค่าเฉลี่ยราคา
- ผลลัพธ์: 171.67
4.3
- เก็บผลลัพธ์รวมเป็น object ที่บอกทั้ง
  - totalPrice = ราคารวมทั้งหมด
  - totalQuantity = จำนวนรวมทั้งหมด
- ผลลัพธ์: { totalPrice: 1080, totalQuantity: 15 }

5.
5.1
- ใช้ .push() เพิ่มสินค้าใหม่เข้าไปท้าย array
5.2
- ใช้ .pop() ลบสินค้าอันสุดท้ายออกจาก array
5.3

5.4

5.5
